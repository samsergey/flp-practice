Полугруппы и моноиды

Загрузите модуль Data.Monoid. Изучите информацию о классе Monoid с помощью команды :t в GHCi. В модуле Data.Monoid для функции mappend определено более котороткое обозначение: <>. Поэкспериментируйте со следующими моноидами: 
 [1,2,3] <> [5,6]
 "Hello" <> " world" <> "!"
 mempty :: String
 Sum 5 <> Sum 10
 mempty :: Sum Int
 Product 3 <> Product 5
 mempty :: Product Int
 Any True <> Any False
 All True <> All False
 С помощью функции foldMap найдите для списка [2,4,-3,10,6]
  • сумму,
  • произведение,
  • все ли элементы чётные,
  • есть ли в списке отрицательные числа
Моноид образует тип Ordering. Выясните что является моноидальной суммой и нейтральным элементом для этого типа. Придумайте, зачам бы это могло пригодиться. Кортеж (пара) моноидов тоже является моноидом. Используя это свойство получите за один проход списка сумму и произведение его элементов. Определите экземпляр класса Monoid для типов Min и Max, соответствующих минимальномуи и максимальному значению аргумента. Включите нейтральные элементы None в описание этих типов. Найдите для заданного списка чисел за один проход сумму и произведение элементов, а также минимальный и максимальный элементы. Определите оператор &&& имеющий тип:
(&&&) :: (a -> b) -> (a -> c) -> (a -> (b,c))
Решите с его помощью предыдущую задачу. Моноид образуют и функции, возвращающие моноид. Удостоверьтесь в этом на нескольких примерах. Напишите определение функции 
when :: Monoid m => (a -> Bool) -> m -> a -> m
Решите задачу FizzBuzz, используя эту функцию и то обстоятельство, что строки образуют моноид. Напишите определение функции wc, подсчитывающую за один проход текста количество строк, слов и символов в нём. Для этого стоит определить новый тип Resumable m, который позволит останавливать и возобновлять накопление результатов.
