Emacs.
Краткая справка по наиболее часто используемым командам

Комбинации клавиш

С-…соответствует нажатию Ctrl + …
M-… соответствует нажатию Alt + …
S-… соответствует нажатию Shift + …
C-… C-… два нажатия подряд (не одновременно)
C-M-… соответствует одновременному нажатию Alt + Ctrl + …
C-g отмена набора текущей команы

Справка

C-h C-h справка о справке
C-h C-m справка о текущих режимах (модах)

Выход из программы

C-x C-c сохранить открытые файлы и выйти.

Управление файлами

C-x C-f открыть или создать новый файл
C-x C-r открыть один из недавно открывавшихся файлов
C-x C-s сохранить текущий файл
С-x C-k закрыть буфер с текущим файлом

Редактирование текста

Кроме стандартных способов перемещения по тексту, есть ряд особых приёмов работы с текстом

M-/ автодополнение слова
M-d удаление слова справа от курсора
M-Delete удаление слова слева от курсора
C-a перейти в начало строки
C-e перейти в конец строки
C-k удаление содержимого строки вправо от курсора
C-SPS начать или завершить выделение региона
C-d удалить символ слева от курсора
M-SPC удалить ряд пробелов, оставив лишь один

Мультикурсоры

Мультикурсоры позволяют вводить текст в нескольких местах одновременно. Если при создании мультикурсора выделен регион, мультикурсоры образуются в регионах, идентичных выделенному. Так удобно переименовывать имена функций или переменных.

C-M-> создаёт мультикурсор ниже по тексту
C-M-< создаёт мультикурсор выше по тексту
RET выход из режима мультикурсора

Инкрементный поиск

С помощью инкрементного поиска очень удобно перемещаться по тексту без использования стрелок или мыши.

С-s инкрементный поиск ниже по тексту
C-r инкрементный поиск ниже по тексту
RET выход из режима инкрементного поиска
C-x C-x возвращение в предыдущую точку в тексте

Повторение команд и их последовательностей

C-x ( начать запись макрокоманды
C-x ) завершить запись макрокоманды
C-x C-e выполнить макрокоманду
C-e повторить выполнение макрокоманды
C-u число … повторение команды … указанное число раз, например, C-u 60 - напечатает 60 знаков '-'
С-u число C-x C-e выполнить макрокоманду указанное число раз

Работа с буферами

C-x 1 скрыть все буферы, оставив открытым только текущий буфер
C-x 2 разделить буфер горизонтально
C-x 3 разделить буфер вертикально
C-x 0 скрыть текущий буфер
C-x b переключиться на один из существующих буферов
C-x o перейти в один из открытых буферов
C-x k уничтожить текущий буфер
C-x } расширить текущий буфер
C-x { сузить текущий буфер

Работа в haskell-mode

C-c C-l скомпилировать программу и загрузить её в интерпретатор
C-c C-z переключаться между буферами программы и интерпретатора
TAB установить отступ сроки (с учётом структуры программы)
C-m перейти в начало строки, игнорируя пробелы
C-/ автодополнение слова
M-; закомментировать строку
M-S-; закомментировать выделенный регион
